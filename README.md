## start server

starting node-red
```bash
$ cd /node-red
$ node red.js
```

starting node red with your own flow from github
```bash
$ cd /node-red
$ node red.js industry/path/to/your/flow.json
```
[more command option](https://nodered.org/docs/getting-started/local#command-line-usage)
